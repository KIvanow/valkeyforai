LIVE - Real Valkey 9 on EC2 + Amazon Bedrock

Live Demos

These demos run real Valkey commands against a live Valkey 9 instance with real Amazon Bedrock embeddings and LLM. Every score, every latency, every response is real.

Vector Search

8 documents embedded with Amazon Titan (1536-dim). Search by meaning and see real cosine similarity scores. Sub-millisecond KNN queries.

FT.CREATE + HSET + FT.SEARCH KNN | Titan Embeddings

Semantic Caching

Ask a question - Claude answers it in ~2 seconds. Ask a similar question - the cached response comes back in ~1ms. See the latency difference in real time.

FT.SEARCH KNN + HSET + EXPIRE | Claude Haiku + Titan Embeddings

Rate Limiting

Click fast and watch yourself get rate limited. Real INCR + EXPIRE pipeline with sub-millisecond latencies and a live latency histogram.

INCR + EXPIRE pipeline | ~0.5ms per request

Conversation Memory

Send messages, switch sessions, load history. Real RPUSH, LRANGE, and EXPIRE with automatic TTL expiry on sessions.

RPUSH + LRANGE + EXPIRE | LIST data structure

Feature Store

Write ML features, read them back, batch-fetch 100 entities in a single pipeline round-trip. See per-entity microsecond latencies.

HSET + HGETALL + Pipeline | HASH data structure
Backend: FastAPI + Valkey 9 (valkey-bundle) on EC2 with Amazon Bedrock for embeddings and LLM.
